# Asterisk IVR with Call Reporting

Production-ready IVR for Asterisk with SQL-based call reports.

## Features
- Custom IVR menu
- Call logging to MySQL/MariaDB
- Queue and agent reporting
- Easy installation
- Asterisk 16+

## Use Cases
- Call centers
- Support hotlines
- Sales IVR systems

## Installation
```bash
chmod +x install.sh
./install.sh
